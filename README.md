# Get ENA Seq

是时候推出针对ENA上测序数据专用的下载工具了。

此工具将[ASCPsra](https://gitee.com/wangshun1121/ASCPsra)中，下载ENA数据的部分剥离出来，并利用ENA数据库的API实现数据信息自动收集整合，最终形成了这一版代码。
